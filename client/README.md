# Repository Tracker - Frontend

Next.js frontend for the Repository Release Tracker. See the [root README](../README.md) for full project documentation.

## Quick Start

```bash
npm install
npm dev
```

## Structure

```
app/
├── components/    # React components
├── lib/           # Utilities and config
├── api/           # API routes (NextAuth)
├── types/         # TypeScript definitions
└── page.tsx       # Main application page
```

## Development

- Built with Next.js
- Uses TailwindCSS for styling
- Apollo Client for GraphQL
- NextAuth.js for authentication

For full setup instructions and documentation, see the [root README](../README.md).
